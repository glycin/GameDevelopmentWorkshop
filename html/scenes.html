Scene Management

In Unity, different levels are called Scenes. They can either be whole levels in your game, or parts of them.

Create Scene

Scenes in unity are essentially what documents are to word. When you try to "Save" by pressing Ctrl-S, it will save the scene you currently have open. If you dont already have a scene open, it will create one. Similarly, a new scene can be created by pressing Ctrl-N or right-cliking on the project folder.

Scenes in the project view

Load Scene

Loading into a new scene is done via the SceneManagement.
Make sure that the new scene is declared in the build settings (File -> Build Settings). 

using UnityEngine.SceneManagement;
// class stuff
void LoadMenu() {
SceneManager.LoadScene("Menu");
}

Load Scene Additive

It is also possible to a scene inside your current scene (instead of switching).
This technique is often used for on-demand features, like character preview.

using UnityEngine.SceneManagement;
// class stuff
void LoadCharacterView() {
SceneManager.LoadScene("CharacterView", LoadSceneMode.Additive);
}

Load Scene Async

Loading scene asynchronously means that it tries to load a scene via a coroutine (separate thread) and giving the main thread some time to keep rendering.
This is used usually for loading screens, although a custom implementation will always work better.
Be aware that it can be buggy!

using UnityEngine.SceneManagement;
// class stuff
void LoadBigScene() {
SceneManager.LoadSceneAsync("BigScene");
}

Quit Application

Quitting the application will close the app completely. 

void Quit() {
Application.Quit();
