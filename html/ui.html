Canvas

Canvas is the parent object that holds all UI related objects.
While you will see it in your scene as if it is placed somewhere in the world (and probably looks huge), it is instead being directly rendered on the camera.

It is important to think of the screen size/aspect ratio of your application when creating the canvas.
Setting the UI Scale Mode in the Canvas Scaler component to Scale With Screen Size will make the canvas stretch so that it matches the screen size of the device.
Setting the reference resolution to 1920x1080, will make sure that the UI always stretches from a default HD 16:9 resolution.
It is important to change this setting before you go on with the UI design! Otherwise, everything else will have a wrong scaling when you make this change later.

Rect Transform

UI Objects use Rect Transform instead of normal Transform.

Rect Transform in the unity inspector gives the option to align the object based on the parent's position.

Panel

A panel is essentially an image that by default stretches 100% by 100% and has lower opacity.
While it is no different from an image, it is listed as a separate object because it serves as a preset for a window background.

A nice way to create two different views in the UI is to have two panels as parent objects.

Image

Image is a UI component that displays a picture, or as it is called in the gaming world, a single sprite.
The source image of an Image component must be of type Sprite. Therefore, after you import an image to unity, you must also set the texture type from the import settings to Sprite (2D and UI). The import settings can be found in the inspector window, by clicking on the image from the project files.


Import settings for a Sprite.


Just like other UI components, an image also aligns based on RectTransform.

Text

The self-explanatory Text component, displays text in the UI.
It can use different fonts (as long as they are imported to Unity first), scale dynamically, change color etc.

Mask

A Mask is a component that hides all underlying content (that are children to the GameObject containing the mask).
The shape of the mask is set by the image component.

World Space

In the canvas component, if the render mode is set to world, then instead of rendering directly on the camera, the canvas will render in the world just like all other objects.

This can be used to create UI inside the world, like computer screens, floating text, dynamic/intricate wall textures etc.
